# **Java-to-C File Transfer**

## Introduction

This project enables **file transfer between Java and C** using **socket programming**. It provides a **Java GUI** for selecting and sending files, while a **C receiver** listens for incoming transfers.

---

## Features

- ✅ **GUI for file selection**
- ✅ **Socket-based file transfer**
- ✅ **Real-time logs in the Java GUI**
- ✅ **Automatic retries & error handling**
- ✅ **Supports large files**

---


## Usage
- Run the Java sender, select a file, and click "Transfer File".
- Start the C receiver to accept and save the file.
- Monitor logs in the Java GUI.
- Click "End Command" to stop the transfer.
